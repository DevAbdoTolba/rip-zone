## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge inner padding, muscle tag gaps |
| sm | 8px | Compact element spacing, filter chip gap, card inner row gaps |
| md | 16px | Default element spacing, card padding, search bar horizontal padding |
| lg | 24px | Section header padding, muscle group section top margin |
| xl | 32px | Layout horizontal margins (page container) |
| 2xl | 48px | Major section breaks between muscle group sections |
| 3xl | 64px | Page-level bottom clearance above bottom nav |

Exceptions:
- Bottom nav height: 64px (3xl) — reserved for nav bar itself
- Main content bottom padding: 64px — ensures last card is not hidden by bottom nav
- Touch targets for bottom nav icons: minimum 44px tap area (WCAG 2.5.5) — implemented with padding to reach 44px even if icon is 24px
- Mini muscle map render size: 120px wide × 200px tall — fixed dimensions in exercise detail Level 2 expand

Source: 8-point scale default; exceptions from CONTEXT.md D-06, CONTEXT.md D-08

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.1 |

Usage guidance:
- **Body (16px/400):** Exercise description excerpt in Level 1 card, warm-up movement instruction text, empty state body
- **Label (14px/400):** Equipment badges, difficulty badges, primary/secondary muscle tag text, filter chip text, bottom nav labels, form cues in Level 2 expand
- **Heading (20px/600):** Exercise name in card header, muscle group section titles (e.g., "Chest", "Back"), warm-up sheet header
- **Display (28px/600):** Page title "Exercises" at top of `/exercises` page

Source: Phase 1 dark-mode-only decision. Typography sizes are defaults (upstream artifacts did not specify); weights match established pattern (Phase 2 used foreground/muted-foreground, no custom weight declarations).

## Color

All colors are OKLCH tokens defined on `:root` in `src/app/globals.css`. Dark mode only — no `.dark` class selector.

| Role | OKLCH Token | Value | Usage |
|------|-------------|-------|-------|
| Dominant (60%) | `--background` | `oklch(0.07 0.01 265)` | Page background, full-bleed surfaces |
| Secondary (30%) | `--card` | `oklch(0.10 0.02 265)` | Exercise cards, warm-up sheet background, filter chip unselected state, bottom nav background |
| Accent (10%) | `--primary` | `oklch(0.85 0.18 195)` | Reserved list below |
| Muted surface | `--muted` | `oklch(0.12 0.02 265)` | Input background for search bar, inactive section backgrounds |
| Muted text | `--muted-foreground` | `oklch(0.50 0.02 250)` | Description excerpt text, secondary label text, placeholder in search |
| Foreground | `--foreground` | `oklch(0.95 0.01 270)` | Primary text — exercise names, headings, active nav label |
| Border | `--border` | `oklch(0.15 0.03 265)` | Card borders, input borders, section dividers |
| Destructive | `--destructive` | `oklch(0.60 0.20 25)` | Not used in this phase — no destructive actions |
| Electric purple | `--secondary` | `oklch(0.60 0.25 290)` | Difficulty badge "Hard" variant only (visual differentiation from primary accent) |

**Accent (`--primary`) reserved for:**
1. Active bottom nav tab indicator and icon
2. Active filter chip selected state (background fill or border)
3. Search bar focus ring
4. Mini muscle map: primary muscle highlight fill (`oklch(0.85 0.18 195)`)
5. Mini muscle map: secondary muscle highlight fill — dimmed variant `oklch(0.55 0.10 195)` (same hue, lower chroma/lightness)
6. "View warm-up" badge/link text color on exercise card
7. Expand "More" button text color (Level 1 → Level 2 trigger)

**Never use accent for:** Unselected chips, card backgrounds, general text, section titles, description text.

Source: `src/app/globals.css` OKLCH tokens (Phase 1 established); mini muscle map accent values from CONTEXT.md D-09; accent reservation list extrapolated from Phase 2 contract (`text-primary` reserved for accents per STATE.md Phase 02 note).

## Interaction Contracts

### Search bar behavior
- Input is sticky at top of page (position: sticky, top: 0, z-index above cards)
- Filters exercise list as user types — no submit button, no debounce (instant filter on every keystroke via `useMemo`)
- Placeholder text: "Search exercises…"
- Clear button (X icon, lucide X) appears inside input when query is non-empty
- When query is active, muscle group sections collapse to show only matching exercises; empty sections are hidden entirely
- Search is case-insensitive substring match on exercise name

### Filter chips behavior
- Equipment chips: Barbell, Dumbbell, Bodyweight, Machine, Cable, Kettlebell, Resistance Band, Other (8 types)
- Difficulty chips: Beginner, Intermediate, Advanced (3 levels)
- Single-select per category (selecting a new chip deselects previous in same category)
- Active chip: `--primary` accent background fill, `--primary-foreground` text
- Inactive chip: `--card` background, `--muted-foreground` text, `--border` border
- Chips are horizontally scrollable on narrow viewports (overflow-x: auto, no wrapping)
- Chips placed below sticky search bar, also sticky (stacks below search input)

### Muscle group section expand/collapse
- Sections start expanded
- Section header is tappable — collapses/expands the exercise list for that group
- Collapse icon: ChevronDown (rotates 180deg when collapsed) — lucide ChevronDown
- When search/filter is active, sections auto-expand to show matching exercises; empty sections hide completely
- Collapse state is not persisted (resets on page navigation)

### Exercise card two-level expand (D-08)
- Cards start collapsed (Level 0: name only visible in list context — i.e., inside section)
- **Level 1 (first tap on card):** Expands to show: exercise name (heading), 1-line description excerpt, primary muscle badges, equipment + difficulty badges, "View warm-up" badge link, "More" button
- **Level 2 (tap "More"):** Expands further inline to show: all form cues (numbered list), secondary muscle badges, mini muscle map silhouette (120×200px), difficulty + equipment restated
- Expand/collapse animation: `animate-accordion-down` / `animate-accordion-up` from `tw-animate-css`
- "Less" button appears at Level 2 to collapse back to Level 1
- Only one card can be at Level 2 at a time (opening Level 2 on a new card closes any other at Level 2)
- Level 1 can have multiple cards open simultaneously

### Mini muscle map (D-09)
- Rendered at 120px wide × 200px tall, centered within Level 2 expand
- Shows front view if primary muscles are front-body muscles; back view if back-body muscles
- Primary muscles filled with `oklch(0.85 0.18 195)` (bright neon cyan)
- Secondary muscles filled with `oklch(0.55 0.10 195)` (dimmed cyan, same hue)
- Non-targeted muscles rendered at `oklch(0.22 0.02 265)` (same as Phase 2 default muscle fill)
- SVG is display-only — no click/hover interactions on mini map
- Muscle ID lookup: slug → `#muscle-{slug}` via `querySelector`, consistent with Phase 2 MuscleMapCanvas pattern

### Warm-up bottom sheet (D-11)
- Triggered by tapping "View warm-up" badge on exercise card (Level 1)
- Opens as bottom sheet covering ~80% of viewport height on mobile
- Desktop: centered modal dialog
- Header: muscle group name (e.g., "Chest Warm-up") in Heading size (20px/600)
- Content: ordered list of 3-5 warm-up movements, each showing: movement name (Label/600), brief instruction (Body/400), duration/reps (Label/muted-foreground)
- Content area: `max-h-[80vh] overflow-y-auto` to prevent overflow issues
- Dismiss: tap backdrop, press ESC, or swipe down (native @base-ui Dialog behavior)
- No secondary action — sheet is informational only

### Bottom navigation (D-06)
- Mobile (below md breakpoint): fixed bar at bottom of viewport, 64px height
- Desktop (md and above): horizontal tab nav at top of page, replaces bottom bar
- 2 tabs in this phase: Map (MapPin icon) and Exercises (Dumbbell icon)
- Active tab: `--primary` accent icon + label, visual indicator (underline or filled pill)
- Inactive tab: `--muted-foreground` icon + label
- Bottom nav background: `--card` (`oklch(0.10 0.02 265)`) with top border `--border`
- Tab labels: 12px (below Label base, appropriate for nav context), weight 400
- Icons: 24px lucide icons, touch target padded to 44px minimum

### URL deep-linking (D-07)
- URL params read on mount to initialize filter state: `?q=bench`, `?equipment=barbell`, `?muscle=pectoralis-major`
- URL is init-only — mid-session filter changes do NOT update the URL (avoids full navigation events on keystrokes)
- Deep links from Phase 6 click-to-muscle panel will use `?muscle={slug}` to pre-filter the library

## States Inventory

Every interactive component must handle all states listed:

### Exercise card
| State | Visual |
|-------|--------|
| Level 0 (collapsed) | Name only, chevron-right indicator |
| Level 1 (expanded) | Full card with badges, description, CTA badges |
| Level 2 (full detail) | Level 1 + form cues + mini muscle map |
| Hover (desktop) | Card border brightens to `--primary/30` |

### Filter chip
| State | Visual |
|-------|--------|
| Inactive | `--card` background, `--muted-foreground` text, `--border` border |
| Active | `--primary` background, `--primary-foreground` text, no border |
| Hover (desktop) | `--muted` background when inactive |

### Search input
| State | Visual |
|-------|--------|
| Empty | Placeholder text in `--muted-foreground` |
| Active (with value) | Clear (X) button appears at right |
| Focus | `--ring` (`--primary`) focus ring |

### Bottom nav tab
| State | Visual |
|-------|--------|
| Inactive | `--muted-foreground` icon and label |
| Active | `--primary` icon and label, accent underline or filled pill indicator |

### Warm-up sheet
| State | Visual |
|-------|--------|
| Closed | Not rendered |
| Opening | Slides up from bottom (tw-animate-css transition) |
| Open | Full sheet visible, backdrop dimmed at `--background/80` |
| Closing | Slides down |

### Muscle group section
| State | Visual |
|-------|--------|
| Expanded (default) | Full exercise list visible, chevron pointing down |
| Collapsed | Exercise list hidden, chevron rotated 180deg |
| Search-active, has matches | Force-expanded, exercises visible |
| Search-active, no matches | Section hidden entirely |
