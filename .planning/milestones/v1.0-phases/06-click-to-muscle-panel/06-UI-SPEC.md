## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge gaps in flex rows |
| sm | 8px | Gap between badges in a row, compact label spacing |
| md | 16px | Panel horizontal padding, section internal padding |
| lg | 24px | Gap between major panel sections (header / strain card / exercise list / warm-up) |

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (normal) | 1.5 |
| Label / Badge | 14px | 400 (normal) | 1.4 |
| Heading (panel muscle name) | 20px | 600 (semibold) | 1.2 |
| Section heading (within panel) | 14px | 600 (semibold) | 1.4 |

Notes:
- 16px body at normal weight matches established `ExerciseCard` pattern (`text-[16px] font-normal`)
- 20px semibold matches Level 1 expanded exercise name in `ExerciseCard` (`text-[20px] font-semibold`)
- 14px used for badges, secondary labels, form cues — matches existing badge usage
- Max 2 weights: 400 and 600. No 500, 700, or other weights.

Source: `src/components/exercise-library/ExerciseCard.tsx` (confirmed existing typography)

## Interaction Contract

### Panel Open/Close

| Trigger | Action |
|---------|--------|
| Tap any muscle on map | `selectMuscle(slug)` — panel opens automatically (selectedMuscle !== null) |
| Tap same muscle again | `selectMuscle(null)` via map — panel closes |
| Tap a different muscle | `selectMuscle(newSlug)` — panel content switches instantly, no close/reopen |
| X button click | `Drawer.Close` triggers `onOpenChange(false)` → `selectMuscle(null)` |
| Swipe down (mobile) | @base-ui Drawer handles gesture → `onOpenChange(false)` → `selectMuscle(null)` |
| Click backdrop (mobile) | @base-ui Drawer backdrop dismiss → `onOpenChange(false)` → `selectMuscle(null)` |
| Toggle front/back view | `useEffect` watching `currentView` → `selectMuscle(null)` (closes stale panel) |

Source: CONTEXT.md locked interaction decisions

### Drawer Root Props

```typescript
<Drawer.Root
  open={selectedMuscle !== null}
  onOpenChange={(open) => { if (!open) selectMuscle(null) }}
  modal={false}
  swipeDirection="left"   // right-side drawer on desktop; bottom sheet ignores this
>
```

Source: `06-RESEARCH.md` Pattern 1 (verified against DrawerRoot.d.ts)

### Collapsible Sections

| Section | Default state | Trigger element |
|---------|---------------|-----------------|
| "Also targets" secondary exercises | Collapsed | Row with ChevronRight — full-width clickable, 44px min tap target |
| Warm-up section | Collapsed | Row with ChevronRight — full-width clickable, 44px min tap target |

Animation: `animate-accordion-down` (tw-animate-css, already used in ExerciseCard)

### Exercise Card in Panel

Panel exercise items are simplified vs ExerciseCard (no Level 2 in panel — panel is not the deep-dive view). Each primary exercise item shows:
- Name at 16px/400
- Equipment badge + difficulty badge (14px)
- Tap to expand: description (capped at 2 lines) appears inline
- No MiniMuscleMap inside panel cards — keep panel lightweight

Source: CONTEXT.md locked decision ("compact card list, tap to expand details — reuses Phase 3 card pattern")

### "View all exercises" Link

- Text: "View all exercises"
- Destination: `/exercises?muscle={selectedMuscle}`
- Tailwind: `text-primary text-[14px] font-normal`
- Placement: below exercise list, above warm-up section divider
- On click: no panel close — navigates away (user leaves home page)

Source: CONTEXT.md locked decision (D-07 deep-linking)

## Content Architecture (Panel Sections)

Content order is locked from CONTEXT.md:

1. **Panel Header**: Muscle display name (20px/600) + strain badge + X close button
2. **Strain Status Card**: Colored badge with level text + mini strain bar (0–100% fill)
3. **Exercise List**: Primary exercises (always visible) + "Also targets" collapsible (secondary exercises)
4. **"View all exercises" link**
5. **Warm-up Section**: Collapsible, shows movements from warmups.json for the muscle's group

```
┌─────────────────────────────────┐
│  [Heavy]  ████████░░  70%       │
└─────────────────────────────────┘
```

- Badge: `text-[14px]` with strain color as background tint (15% opacity) and strain color as text

## Accessibility Contract

|---------|-------------|
