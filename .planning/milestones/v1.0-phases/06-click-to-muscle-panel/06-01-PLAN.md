**2. PanelExerciseList.tsx**

Props: `{ exercises: ExerciseJSON[], selectedMuscle: string, muscles: MuscleJSON[] }`

Where `ExerciseJSON` and `MuscleJSON` are inline types matching the JSON shapes (or use `any` with documented shape — the data is from JSON import).

Use `useMemo` with `selectedMuscle` dependency to split exercises:
- `primaryExercises`: `exercises.filter(ex => ex.primaryMuscles.includes(selectedMuscle))`
- `secondaryExercises`: `exercises.filter(ex => ex.secondaryMuscles.includes(selectedMuscle) && !ex.primaryMuscles.includes(selectedMuscle))`

State: `const [expandedSlug, setExpandedSlug] = useState<string | null>(null)` for exercise item expand, `const [showSecondary, setShowSecondary] = useState(false)` for collapsible section.

Reset both states when `selectedMuscle` changes via `useEffect(() => { setExpandedSlug(null); setShowSecondary(false) }, [selectedMuscle])`.

Render:
- Section heading: `<h3 className="text-[14px] font-semibold text-foreground mb-2">Exercises</h3>`
- If `primaryExercises.length === 0`: show `<p className="text-[14px] text-muted-foreground">No exercises found for this muscle group.</p>`
- Else: `<ul className="space-y-2">` with each exercise as a `<li>`:
  - Collapsed: name (16px/400) + equipment badge + difficulty badge in a row. ChevronRight icon. Full `<li>` is clickable (min-h-[44px]).
  - Expanded (when `expandedSlug === ex.slug`): name at 16px/400, description capped at 2 lines (`line-clamp-2`), equipment + difficulty badges. ChevronDown icon. Uses `animate-accordion-down` from tw-animate-css.
  - On click toggle: `setExpandedSlug(prev => prev === ex.slug ? null : ex.slug)`

Equipment labels map: `{ barbell: 'Barbell', dumbbell: 'Dumbbell', bodyweight: 'Bodyweight', machine: 'Machine', cable: 'Cable', kettlebell: 'Kettlebell', band: 'Band', other: 'Other' }`
Difficulty labels map: `{ beginner: 'Beginner', intermediate: 'Intermediate', advanced: 'Advanced' }`

- "View all exercises" link after exercise list: `<Link href={'/exercises?muscle=${selectedMuscle}'} className="text-primary text-[14px] font-normal inline-block mt-3">View all exercises</Link>` — import `Link` from `next/link`.

- Secondary exercises collapsible:
  - Only render if `secondaryExercises.length > 0`
  - Trigger row: `<button onClick={() => setShowSecondary(s => !s)} aria-expanded={showSecondary} className="flex items-center justify-between w-full py-3 min-h-[44px] text-[14px] font-semibold text-foreground">Also targets this muscle {showSecondary ? <ChevronDown size={16} /> : <ChevronRight size={16} />}</button>`
  - Content: same `<ul>` pattern as primary, wrapped in `{showSecondary && <div className="animate-accordion-down">...</div>}`

Add `data-testid="panel-exercise-list"` on root div.

- If `movements.length === 0`: show `<p className="text-[14px] text-muted-foreground py-2">No warm-up guidance available for this muscle.</p>` inside the expanded content.
