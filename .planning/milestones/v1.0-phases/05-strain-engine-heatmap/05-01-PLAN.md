<feature>
  <name>Strain Engine — Pure Computation Function</name>
  <files>src/lib/strain-engine.ts, tests/lib/strain-engine.test.ts</files>
  <behavior>
    - computeStrainMap([single fresh chest dose at volume 5000], now) returns Map with "pectoralis-major" => StrainLevel.Strained
    - computeStrainMap([chest dose completedAt 72hrs ago], now) returns roughly half the strain — expect StrainLevel.Moderate or lower depending on initial volume
    - computeStrainMap([chest dose completedAt 144hrs ago (6 days)], now) returns ~25% strain — expect StrainLevel.Light or Rested
    - computeStrainMap with primary muscle (multiplier 1.0) at volume 3000 produces higher strain than secondary (multiplier 0.4) at same volume
    - computeStrainMap with extreme volume (50000) still caps at StrainLevel.Strained (never exceeds 100%)
    - computeStrainMap with empty doses array returns empty Map
    - computeStrainMap with multiple sessions accumulates strain — two moderate sessions produce higher strain than one alone
    - Rested muscles (below Light threshold) are NOT added to the result Map
    - STRAIN_COLORS maps each StrainLevel to its hex/oklch color: Light=#3b82f6, Moderate=#eab308, Heavy=#f97316, Strained=#ef4444
  </behavior>
  <implementation>
    Create src/lib/strain-engine.ts as a pure function with zero external I/O dependencies.

    WorkoutMuscleDose interface:
    - muscleSlug: MuscleSlug — which muscle was worked
    - volume: number — sets x reps x weightKg (pre-calculated by caller)
    - multiplier: number — 1.0 for primary muscles, 0.4 for secondary
    - completedAt: number — Unix ms timestamp of session completion

    Constants:
    - HALF_LIFE_MS = 72 * 60 * 60 * 1000 (72 hours)
    - NORMALIZE_DIVISOR = 5000 (tuning constant: ~5 sets x 10 reps x 100kg = "full strain at moment of completion")
    - Thresholds: Light=20, Moderate=40, Heavy=60, Strained=80 (percent of normalized 0-100 scale)

    Algorithm:
    1. For each dose, compute decayed strain: (volume * multiplier) * Math.pow(0.5, (now - completedAt) / HALF_LIFE_MS)
    2. Accumulate decayed strain per muscleSlug into a raw Map
    3. Normalize: pct = Math.min(100, (rawSum / NORMALIZE_DIVISOR) * 100)
    4. Map pct to StrainLevel using threshold boundaries
    5. Only include non-Rested muscles in result Map

    Also export STRAIN_COLORS constant:
    ```typescript
    export const STRAIN_COLORS: Record<StrainLevel, string> = {
      [StrainLevel.Rested]:   'oklch(0.22 0.02 265)',
      [StrainLevel.Light]:    '#3b82f6',
      [StrainLevel.Moderate]: '#eab308',
      [StrainLevel.Heavy]:    '#f97316',
      [StrainLevel.Strained]: '#ef4444',
    }
    ```
  </implementation>
</feature>
